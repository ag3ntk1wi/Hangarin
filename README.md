# HANGARIN
Welcome to my simple web application, Hangarin!
***
## Short Description
Hangarin helps users organize their daily tasks, manage priorities, add notes, and break down large goals into smaller subtasks.
***
## List of Features  
* Priority Section
* Category Section
* Task Section
* Note Section
* SubTask Section
